## Development

This project uses [racket](https://racket-lang.org/) and [pollen](https://docs.racket-lang.org/pollen/) to build the various implementations,
and the website.

### Setup

Install dependencies:

``` sh
make install-deps
```

Install the core `earl-grey` package (this repo):

``` sh
make install
```
